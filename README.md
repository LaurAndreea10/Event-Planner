# Event Planner

A modern event planning front-end project built as part of a personal portfolio, designed to organize information in a clear, structured, and user-friendly way.

## Live Demo

https://laurandreea10.github.io/Event-Planner/

## Features

* Event planning interface
* Organized content and workflow structure
* Interactive front-end layout
* Responsive design
* Modern visual presentation
* GitHub Pages deployment

## Technologies

* HTML
* CSS
* JavaScript
* GitHub Pages

## Purpose

This project was created as a personal front-end experiment exploring how event-related information can be presented in a clean, intuitive, and visually appealing digital format.
